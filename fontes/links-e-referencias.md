# 📚 Fontes e Referências

Lista completa das fontes utilizadas no caderno temático do NotebookLM.

## Fonte 1 — Python Data Science Handbook

- **Autor:** Jake VanderPlas
- **Tipo:** Livro (disponível gratuitamente online em HTML e PDF)
- **Link:** https://jakevdp.github.io/PythonDataScienceHandbook/
- **Por que escolhi:** É considerado uma das melhores referências gratuitas para iniciantes em Ciência de Dados com Python. Cobre NumPy, Pandas, Matplotlib e Scikit-learn de forma progressiva.
- **Como usar no NotebookLM:** Copie o conteúdo dos capítulos desejados em formato texto, ou baixe o PDF pelo GitHub do autor: https://github.com/jakevdp/PythonDataScienceHandbook

## Fonte 2 — Documentação Oficial do Pandas

## 💡 Dicas para Curadoria de Fontes no NotebookLM

1. **Prefira PDFs** — o NotebookLM processa melhor documentos PDF do que links diretos
2. **Limite o tamanho** — documentos muito grandes podem perder contexto; prefira capítulos específicos
3. **Misture os tipos** — combine livros técnicos, tutoriais práticos e artigos para ter perspectivas diferentes
4. **Documente a origem** — sempre anote de onde veio cada fonte para referenciar corretamente
5. **Verifique a data** — para tecnologia, prefira fontes dos últimos 3-5 anos
